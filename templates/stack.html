Tecnologias y lenguajes de programación que aprendimos

Somos la generación 2022 - 2023 de código futuro

  • Visual Studio Code, HTML, CSS, BOOTSTRAP

    Visual Studio Code: Versátil editor para desarrollar código. HTML: Lenguaje base web. CSS: Estiliza páginas. Bootstrap: Framework CSS, facilita y acelera desarrollo web adaptable con componentes preestablecidos y diseño responsivo, optimizando la experiencia del usuario en diferentes dispositivos.

  • PYTHON - ESTRUCTURA DE DATOS, MODULARIZACIÓN

    Python: Lenguaje ágil y legible. Destaca en estructuras de datos eficientes. Modularización simplifica proyectos al dividir en módulos independientes, fomentando la reutilización y mantenimiento del código para escalabilidad.

  • TERMINAL, GIT, GITHUB

    Terminal: Interfaz de línea de comandos. Git: Sistema de control de versiones. GitHub: Plataforma para alojar proyectos Git. Juntos, permiten gestionar código, versionarlo y colaborar en desarrollo de software mediante seguimiento, control y almacenamiento de proyectos.

  • APIS CON POSTMAN, FLASK

    Postman: Herramienta para probar y desarrollar APIs. Flask: Framework de Python para construir aplicaciones web. Combinados, permiten crear y probar fácilmente APIs web, simplificando el desarrollo y pruebas de aplicaciones.
